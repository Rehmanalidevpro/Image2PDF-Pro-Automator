== 📖 Overview
Developed by *Rehman Ali*, the **Image2PDF Pro Automator** is an elite CLI-based engine designed to transform your raw images into professionally structured PDF documents. No more messy file names or disorganized pages—everything is automated with precision.

toc::[]

== 🔥 Premium Features
[cols="1,3", frame=none, grid=none]
|===
| ⚡ *Smart Engine* | Natural numeric sorting (1, 2, 10...) prevents sequencing errors.
| ✍️ *Auto-Stamp* | Automatically injects aesthetic page numbers at the bottom of each page.
| 🛠️ *Self-Healing* | Detects and installs missing dependencies on the fly.
| 🎨 *Pro UI* | Colorful terminal interface for a better user experience.
| 🛡️ *Safe-Guard* | Zero risk to original files; all operations happen in isolated output folders.
|===

== 🚀 Installation & Launch

[IMPORTANT]
====
Ensure you have Python 3.x installed. The tool handles the rest!
====

. *Clone the repository:*
[source,bash]
----
git clone https://github.com/Rehmanalidevpro/Image2PDF-Pro-Automator.git
----

. *Navigate & Execute:*
[source,bash]
----
cd Image2PDF-Pro-Automator
python pdf_pro_automator.py
----

== 🛠 Usage Workflow
[horizontal]
Step 1 :: Organize your images with numeric names (e.g., `1.jpg`, `2.jpg`).
Step 2 :: Run the script and paste your folder path.
Step 3 :: Set your custom PDF name.
Step 4 :: Access your file in the `PDF_Output_Folder`.
